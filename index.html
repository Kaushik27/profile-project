What's inside each folder

Every design is self-contained: index.html, styles.css, script.js. No build step, no dependencies — drop it on any static host.

To preview locally: open the design's index.html directly in your browser, or run python3 -m http.server 8000 from this folder and visit localhost:8000.

To pick a winner: copy that subfolder's files to your own repo, or just deploy it. See README.md for deploy + git instructions.
